Intransitive Pp c

IRI http://www.lexinfo.net/lexinfo#IntransitivePP
Description

A class representing a subcategorization frame for an intransitive verb subcategorizing a specific prepositional object, e.g. "a river (subj) flows through a city (prepositional object)". Every subcategorization frame of type intransitivePP has exactly two syntactic arguments: one subject and one prepositional object.
Sub Class Of lmf:SubcategorizationFrame c
Restriction
  • lmf:hasSyntacticArgument op exactly 2
  • lmf:hasSyntacticArgument op some
  • lmf:hasSyntacticArgument op some

Noun Pp c

IRI http://www.lexinfo.net/lexinfo#NounPP
Description

NounPP represents a nominal subcategorization frame in which a noun subcategorizes a certain prepositional object, as in "capital of". In many cases such nominal subcategorization frames are realized in the form of a copula construct, e.g. "Berlin is the capital of Germany", such that we distinguish between Subject as syntactic argument ("Berlin" in the example) as well as the prepositional object ("Germany" in the above case).
Sub Class Of lmf:SubcategorizationFrame c
Restriction
  • lmf:hasSyntacticArgument op some
  • lmf:hasSyntacticArgument op exactly 2
  • lmf:hasSyntacticArgument op some
